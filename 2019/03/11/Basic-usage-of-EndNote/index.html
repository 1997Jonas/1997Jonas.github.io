Basic usage of EndNote/EndNote的基本使用

Basic Skills/基本技能 2019/03/11

Introduction

EndNote is a software developed by ISI Thomson. Its main features include:

  • Collect and store the documents. (Title, arthor, year, journal, reference type, etc.)
  • Find and manage the documents. (Search, rating, group, etc.)
  • Help the researchers use the correct format for the papers. (Paper templates for different journals, working together with Word, etc.)

We can regard what EndNote does as building a personal library, where the books, journals, videos, legal papers, pictures and others are stored.

Besides EndNote Destop, EndNote Web is also availbale.

Building or opening a libary

File-> New… or File-> Open Library… or Open recent

Tip:
If you want share or move your labrary by copying the files, you need to copy both the .data file and the .enl file. They need to work togeter.

Importing Documents

  1. Manual input(Not recommanded):
    Reference -> New Reference or click the ‘New Reference’ button or use the keyboard shortcut (Ctrl+N).

    When you input the author, the system will regrad the last word as the family name automatically. And if the arthor’s given name contains more than one word, the first name and the last name should be seperated by ‘,’.

    Remember to attach the files (.pdf, .jpg, .mov, etc.), if not, it’s just a bunch of information. Use ‘the find full text’ function can make it easy to get done.

    Steps:
    Choose the document you want to find the full text, and then click the Reference-> Find full text. Make sure you have right network connect, which means you can get access to the databases.

  2. Importing file…/Folder…:
    File->Import->File…/Folder….

    When you are importing a file, remember to change the import option to DPF. Besides, you can import a library here just by change the import option to EndNote Library.

    When you are imorting a folder, you can choose if the result includes the subfolders and create a group set for this import.

  3. Importing online:
    Take Web of Science as an example.

    • Choose the papers need downloading.

    • Choose save to EndNote Destop (Or EndNote Online if you like).

    • Choose the content to be saved and click send button.

    • Open the .ciw file that is just downloaded by EndNote Destop.

    • Find full text for the papers or attach them manually.

    Likely, this approach can be used in Google Scholar, 万方数据库, etc.

    More details about how to import documents from diffrent databases or connect other libraries can be seen in EndNote & Word文献管理与论文写作 第2版_童国伦主编2014.

Managing the Libraries

  1. Group:

    • Set up groups: To classify the documents.

      • Right-click the ‘My Groups’ in the left of the UI.
      • Groups->Create Group

      Smart Group:
      Assemble the documents that contain the same key words.

    • Create groups from groups.
      Create From Groups: Intersection, complement, etc.

    • Create groups set.
      The directory above the groups level.

  2. Search the documents.

    • Quick search.
    • Search Panel. Click the button to use.

  3. Find the Duplicates.

    • References->Find Duplicates
    • The setting of this function can be changed in Edit->Preference->Duplicates

  4. Revise the ducuments’ data (Title, abstract, etc.)
    Edit->Find and Replace

  5. Back-up for the libraries.

    • Copy and paste the .data and .enl files.
    • File->Save a Copy…

  6. Compress the Library
    Compressed Library(.enlx)…

  7. Filter.
    Filters need to be created or found online when the embedded filters are not enough, more details is in EndNote & Word文献管理与论文写作 第2版_童国伦主编2014.

  8. Change the field(字段) to be shown about a document.
    Edit->Preference->Display Fields

Paper Writing

  1. Papers templates.
    Tools->Manuscript templates.

  2. Cite While You Write (CWYW).
    To use this function, you need to open both EndNote and Word.

    • Drag the papers you want to cite in EndNote to Word.
    • Click the ‘Update Citations and Bibliography’ in EndNote X9 (or other versions) in Word.
